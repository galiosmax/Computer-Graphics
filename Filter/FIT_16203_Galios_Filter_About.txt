--Описание программы--
В toolbar добавлены кнопки:
1. Создание нового файла - очищает все три поля.
2. Открытие файла - открывает bmp/png файлы.
3. Сохранение файла - сохранение картинки из поля С.
4. Сохранение файла как.
5. Закрытие программы. 
6. Select - поле А становится активным, можно выделять часть картинка 350*350, она будет помещаться в поле Б.
7. Из Б в С - копирует картинку из Б в С, реализован drag'n'drop.
8. Из С в Б - то же самое.
9. Черно-белое изображение - если в поле Б есть картинка, то в С появляется черно-белая ее версия.
10. Негатив.
11. Дизеринг Флойда-Стейнберга.
12. Упорядоченный дизеринг.
13. Зум - реализован зум в произвольное целое число раз. Алгоритм билинейной интерполяции.
14. Оператор Робертса.
15. Оператор Собеля.
16. Блюр - реализовано размытие по Гауссу.
17. Резкость.
18. Тиснение.
19. Акварелизация.
20. Поворот на угол от -180 до 180
21. Гамма-коррекция.
22. Кнопка About.

Для всех кнопок в меню и в toolbar добавлены иконки.
При закрытии программа спрашивает о сохранении только если в поле С есть картинка.
